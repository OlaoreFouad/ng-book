ngSwitch This directive works like the common switch-case construct present in some programming languages. Where a variable is matched against different values (cases), and when a match is found, it performs a corresponding operation. It is a neater alternative to multiple ngIf expressions. It also provides a convenient medium to display specific elements if no matches are found.
